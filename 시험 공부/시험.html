form 태그의 속성 1) action : 내부에 작성된 input 값을 어디로 전달할지 지정하는 속성 -> 주소를 작성하게 됨. 2) method(방식) : 데이터를 서버로 전달할 때의 방식을 지정하는 속성 3) name = form 태그의 이름 지정(JS 사용) 4) target : action에 지정된 주소를 새 탭/현재 탭 중 어디서 열지 지정 button 태그 : 버튼 모양을 출력하는 태그 - type 종류 1) submit : form태그 내부 input태그 값을 action 주소에 제출하는 버튼 2) reset : form태그 내부 input태그 값을 초기화(처음 상태로 돌림) 3) button : 아무 기능 없는 버튼(JS 이벤트용으로 주로 사용) 기타 입력 관련 태그(textarea / select, option) textarea : 여러 줄을 입력하기 위한 입력 상자 select, option : 선택 상자(select)를 열어서 특정 값(option)을 선택 fieldset / legend fieldset : 테두리를 만들어 그룹을 구분하는 영역 legend : fieldset에 붙여주는 제목
